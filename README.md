## Configurasi

1. Buat .env

```
# Untuk wallet tunggal
PRIVATE_KEY=pk_wallet_1

# Multi Wallet
PRIVATE_KEY_1=pk_wallet_1
PRIVATE_KEY_2=pk_wallet_2
PRIVATE_KEY_3=pk_wallet_3
```

2. Proxy

```
http://username:password@ip:port
http://ip:port
socks5://username:password@ip:port
```

## Penggunaan

jalankan bot dengan:

```bash
node index.js
```
